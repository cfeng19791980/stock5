# stock5 Autoresearch 优化报告

## 关键发现

### 🔴 严重问题：评分与实际表现倒挂！

| 评分区间 | 上涨率 | 平均收益 |
|:--------:|:------:|:--------:|
| 0-30分   | 48.5%  | +40.11%  |
| 30-40分  | 43.4%  | **-30.44%** |
| 40-50分  | 46.4%  | +2.34%   |
| **50-60分** | **53.6%** | **+85.80%** ✅ |
| 60-70分  | 40.2%  | -3.58%   |
| 70+分    | 36.6%  | **-46.79%** |

**问题**：高分区间(60+)反而上涨率更低！这说明模型评分没有正确校准。

## 优化建议

### 1. 调整买入阈值
- **当前**: BUY_THRESHOLD = 54 (normal), 65 (conservative)
- **建议**: 改为 **50-55分** 区间（50-60分区间表现最佳）
- **原因**: 50-60分区间上涨率53.6%，平均收益+85.80%

### 2. 修正评分校准问题
- 调查为什么高分区间预测准确率反而低
- 可能原因：模型过度自信，把不确定的预测打高分

### 3. 信号策略优化
- **hold** 信号平均收益 +66.12%，样本量1485次，可靠
- **反向买入机会** 在震荡市表现好
- **强烈买入** 样本少但收益高(+104%)

### 4. 股票池优化
- 表现最好的: 600183.SH (100%准确率), 688519.SH (94.9%)
- 表现最差的: 688233.SH (55.6%), 603986.SH (59.2%)

## 实验计划

1. 买入阈值从54→52测试
2. 排除70+分的高风险信号
3. 增加50-60分区间的信号权重
